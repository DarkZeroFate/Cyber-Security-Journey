# Cyber-Security-Journey
i'm 24, Non-Native speaker got a bussines management bachelor degree and trying to expand my knowledge and skillset in cyber Security
and this was my journey to learn Cyber Security from scratch

I'm currently working on transitioning into cybersecurity. Since I'm juggling a busy work schedule, I'm using this repository to document my progress, and the occasional breakdown of my virtual machines.

This isn't a collection of perfect tutorials. It's my personal scratchpad—a place to keep track of my home lab, my notes on tools like Splunk or Sysmon etc, and the things I'm learning along the way.

Why this exists
I realized that reading about security is one thing, but actually "doing" it is entirely different. I'm building this to prove to myself (and hopefully, future hiring teams) that I can take a concept, build it, break it, and fix it.

What's in here?
- Lab Setup: The messy (but functional) details of my VM environment.
- Projects: Small, hands-on experiments (like malware traffic analysis).
- Notes: Random things I've learned that I don't want to forget.

If you're also starting out, feel free to look around. I'm learning by doing, so expect things to be a bit "work-in-progress.
